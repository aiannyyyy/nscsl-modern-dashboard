# NSCSL Modern Dashboard

A full-stack dashboard application built with a modern web stack.  
This project showcases real-world frontend and backend integration, RESTful APIs, authentication, and a scalable project structure.

---

## Tech Stack

### Frontend
- React
- TypeScript
- Vite
- Tailwind CSS

### Backend
- Node.js
- Express
- MySQL
- JWT Authentication
