NSCSL Modern Dashboard

A full-stack dashboard application built with a modern web stack.
This project showcases real-world frontend and backend integration, RESTful APIs, authentication, data visualization, and a scalable project structure.

Tech Stack
Frontend

React

TypeScript

Vite

Tailwind CSS

Recharts (data visualization & charts)

React Query (TanStack Query) (server-state management & caching)

Custom React Hooks (logic reuse & cleaner components)

html2canvas (exporting dashboard views as images)

xlsx (Excel import and export functionality)

Backend

Node.js

Express

MySQL

Oracle Database (Oracle SQL queries)

Multer (file upload handling)

JWT Authentication
