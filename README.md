# NSCSL Modern Dashboard

A full-stack dashboard application built with a modern web stack.  
This project showcases real-world frontend and backend integration, RESTful APIs, authentication, data visualization, and a scalable project structure.

---

## Tech Stack

### Frontend
- React
- TypeScript
- Vite
- Tailwind CSS
- Recharts
- React Query (TanStack Query)
- Custom React Hooks
- html2canvas
- xlsx

### Backend
- Node.js
- Express
- MySQL
- Oracle Database
- Multer
- JWT Authentication
